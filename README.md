# 🛒 NovaCart – Premium E-Commerce Website

NovaCart is a modern, responsive, and interactive e-commerce website built using **HTML5, CSS3, and JavaScript**. It provides a clean shopping experience with dynamic product loading, live search, category filtering, shopping cart, wishlist, dark mode, and responsive design.

---

## ✨ Features

- 🛍️ Modern & Responsive UI
- 🔍 Live Product Search
- 📂 Category Filtering
- ❤️ Wishlist Functionality
- 🛒 Shopping Cart
- 🌙 Dark Mode
- 📱 Mobile Friendly Design
- ⚡ Smooth Animations
- 📦 Dynamic Products (JSON)
- 📧 Contact Information
- 📍 Professional Footer

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts
